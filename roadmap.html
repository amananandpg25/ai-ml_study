Burnout Prevention Strategy

⚠️ Burnout is real — plan for it:
1. Never study more than 6 hours in one day. Quality > Quantity.
2. Every 2 weeks, take 1 full rest day — no screens, no code.
3. If you feel overwhelmed, switch to "MVP mode" — just 2 hrs/day for 3 days.
4. Celebrate small wins: first Python script, first model trained, first API call.
5. Hinglish tip: "Ek dum perfect nahi hona — sirf consistent rehna hai." (You don't need to be perfect — just stay consistent.)

Recovery Strategy — If You Fall Behind

📅 Behind by 1 week? Skip the "nice to know" topics for that week (data viz extras, SQL advanced joins) and focus on core concepts only.
Behind by 2 weeks? Compress Month 1 weeks 3–4 into 5 days (2 topics/day), skip optional exercises.
Behind by 1 month? Drop Reinforcement Learning and GAN chapters — they are least critical for initial job hunting. Prioritize Python → ML → Deep Learning → LLMs → RAG → Deployment.

Beginner Mistakes to Avoid

  • Tutorial hell: Watching videos without coding. Always type the code yourself.
  • Skipping math: You don't need calculus on Day 1, but don't skip stats/probability entirely.
  • Not using Git: Commit every project from Day 1. Your GitHub IS your resume.
  • Ignoring deployment: A model no one can use is not complete. Learn Flask early.
  • Copying Kaggle notebooks: Study them, then re-write from scratch without looking.
  • Skipping projects: Projects are more important than 10 extra video hours.
  • Using ChatGPT to write all code: Use it to explain concepts, not to generate your assignments.

Key Preprocessing Concepts (Interview Questions)

ConceptSimple Explanation (Hinglish)
Normalisation vs StandardisationNormalisation = 0 to 1 range mein laana (MinMax). Standardisation = mean 0, std 1 banana (Z-score). Distance-based models ke liye scaling zaroori hai.
One-Hot vs Label EncodingLabel Encoding = numbers assign karna (cat=0, dog=1). Problem: model sochta hai dog > cat. One-Hot = separate column har category ke liye. Nominal data ke liye One-Hot use karo.
IQR Outlier RuleQ3 - Q1 = IQR. Outlier: < Q1 - 1.5*IQR or > Q3 + 1.5*IQR

📌 Key Interview Concept — Precision vs Recall:
Cancer detection: High Recall matters (don't miss cases) → lower Precision okay.
Spam filter: High Precision matters (don't mark real email as spam) → lower Recall okay.
F1 is the balance between both. Use F1 when classes are imbalanced.

Interview Preparation Roadmap

Comprehensive interview prep for AI/ML roles.

Top 25 ML Interview Questions

#QuestionKey Answer Points
1What is the bias-variance tradeoff?Bias = underfitting, Variance = overfitting. Model complexity increases variance. Regularisation reduces variance.
2Precision vs Recall — when to prefer which?Precision for false positive cost (spam). Recall for false negative cost (cancer). F1 for balance.
3How does gradient descent work?Compute gradient of loss w.r.t. weights. Update: w = w - lr × gradient. Repeat until convergence.
4Why use Random Forests over a single Decision Tree?Ensemble reduces variance. Bootstrap sampling + feature randomness → decorrelated trees → lower variance.
5What is regularisation and why do we need it?Penalty on model complexity. L1 (Lasso) for sparsity/feature selection. L2 (Ridge) for weight decay.
6Explain the kernel trick in SVMs.Map data to higher dimension where it's linearly separable, without explicitly computing the mapping. RBF kernel computes similarity.
7What is cross-validation?Split data into K folds. Train on K-1 folds, test on 1. Repeat K times. Average score = unbiased estimate.
8How does backpropagation work?Chain rule applied backwards through network. Compute dL/dw for each weight. Update with gradient descent.
9What is attention mechanism in transformers?Q, K, V matrices. Score = softmax(QK^T/√d) × V. Self-attention: every token attends to every other token.
10What is RAG and why is it needed?Retrieval Augmented Generation. Grounding LLM in external knowledge to reduce hallucination and use updated info.
11Explain gradient vanishing in RNNs.Gradients multiplied many times through BPTT → exponentially small. LSTM solves with cell state and gates.
12What is batch normalisation?Normalise activations of each layer to mean=0, std=1. Reduces internal covariate shift, allows higher learning rates.
13How does K-Means work and what is its complexity?Assign to nearest centroid → update centroid → repeat. O(nkdi) per iteration. Sensitive to init and K choice.
14What is transfer learning?Use weights from model trained on large dataset (ImageNet, web text) and fine-tune on smaller task-specific dataset.
15Explain the difference between GANs and VAEs.GAN: adversarial game, no explicit latent space. VAE: probabilistic encoder-decoder, smooth latent space, KL divergence loss.

Reading Research Papers — Getting Started

How to read a paper (3-pass method):
Pass 1 (5 min): Read title, abstract, conclusion, figures only. Decide if relevant.
Pass 2 (30 min): Read introduction, methods summary, results. Skip equations.
Pass 3 (2 hrs): Read everything. Reproduce key results in code.

Start with these landmark papers: Attention Is All You Need (2017) · BERT (2019) · GPT-3 (2020) · InstructGPT (2022) · ReAct (2022) · RAG Paper (2020) · LoRA (2021)
